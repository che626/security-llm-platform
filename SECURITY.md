# Security Policy

## Scope

This project is a local demonstration platform for defensive SOC workflows:

- log analysis
- RAG-assisted security knowledge lookup
- IOC extraction
- ATT&CK mapping
- SOAR playbook generation in simulation mode

It is not a production SOC platform and should not be exposed to the public internet without additional hardening.

## Reporting Issues

If you find a vulnerability or unsafe behavior, please open a GitHub issue with:

- affected module or endpoint
- reproduction steps
- expected safe behavior
- suggested fix, if available

Do not include real secrets, production logs, private IP inventories, or confidential incident data in public issues.

## Demo Credentials

The included accounts are local demo credentials only. They are intentionally documented for reproducible local testing and must not be reused in production.
