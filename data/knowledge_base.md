# 安全知识库

## SSH 暴力破解

特征：

同一 IP 在短时间内多次登录失败，
尝试 root/admin/test 等用户名。

处置建议：

- 封禁来源 IP
- 检查是否存在成功登录
- 启用 Fail2Ban
- 强制密钥登录
- 禁止 root 远程登录

---

## SQL 注入

特征：

URL 中出现：

- ' OR '1'='1
- UNION SELECT
- information_schema
- sleep(
- benchmark(

等关键字。

处置建议：

- 检查 Web 日志
- 修复参数化查询
- 加入 WAF 规则
- 检查数据库是否泄露

---

## 端口扫描

特征：

同一来源 IP
短时间访问多个端口：

- 22
- 80
- 443
- 3306
- 6379

等。

处置建议：

- 封禁扫描源
- 关闭不必要端口
- 检查暴露服务
- 增加 IDS/IPS 规则

---

## 可疑 C2 回连

特征：

主机周期性访问陌生域名或 IP，
存在：

- POST 请求
- 短周期连接
- 异常 User-Agent
- gate.php
- shell.php

等行为。

处置建议：

- 隔离主机
- 检查进程
- 检查启动项
- 检查计划任务
- 阻断域名和 IP
